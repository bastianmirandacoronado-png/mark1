🔗 Integración Mark 1 — Ejemplo iframe

Agrega el parámetro ?embed=1 a la URL para ocultar el header y nav del dashboard. Útil para integrar vistas específicas en otros proyectos.

Código de ejemplo:

<iframe
  src="http://TU-URL/MARK1/index.php?embed=1"
  width="100%"
  height="800px"
  style="border:none;border-radius:8px"
  title="Dashboard DSSO Mark 1"
></iframe>

Reemplaza TU-URL con la IP local o URL de Cloudflare.
